# Simple Diffusion

Implementation of Denoising Diffusion Probabilistic Models (DDPMs) in JAX.

 
This project works through parts the tutorial material from [A Beginner's Friendly Introduction to Diffusion Models in JAX](https://axeldonath.com/jax-diffusion-models-pydata-boston-2025/) by Axel Donath, presented at PyData Boston 2025.
